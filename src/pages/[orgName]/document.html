Production Environment Notice

Modifications to the web app content, look, and feel need to be done in the development mode. Follow the following steps to upload the content for the production environment:

  • Stop the server and start the development environment using the command: MODE=DEV docker compose up
  • Make the necessary modifications to display the organization content according to the organization's theme.
  • Update the orgContent.json file in the resources folder to reflect your organization information.
  • Once satisfied with the modifications, click the upload icon in the navigation bar.
  • The next page will display the organization information.
  • Click the upload button to upload the content to the production environment.
